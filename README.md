# secretSanta
Secret Santa Project
# Requirements
```
Node >= 18.15.0 
```
# Run Project 
```
git clone git@github.com:NiteshSingh17/secretSanta.git
cd scratch-start
npm run scratch-start
```
open [http://localhost:3000/](http://localhost:3000/)
# Other Commands
# install dependency for Frontend and Backend
```
npm run installDep
```
# Run Tests 
```
npm run test
```
# Start 
```
npm run run-start
```
